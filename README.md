# Simple Interest Calculator

A simple Python program to calculate **Simple Interest**.

## Description

This project calculates Simple Interest using the standard formula:

```
Simple Interest (SI) = (Principal × Rate × Time) / 100
```

Where:
- **Principal (P)** – The initial amount of money
- **Rate (R)** – Annual interest rate (in percentage)
- **Time (T)** – Time period (in years)

## Files

- `simple_interest.js` – Main JavaScript script for calculating simple interest

## Usage

Run the script using Node.js:

```bash
node simple_interest.js
```

## Example

```
Principal: 1000
Rate: 5%
Time: 3 years
Simple Interest: 150
Total Amount: 1150
```

## Formula

```
SI = (P × R × T) / 100
Total Amount = P + SI
```
